Once you got it on you want to select create custom profile to get mods separated from the modpacks. this will allow you to get random mods from different versions (it will vary on some versions the mod is only on).

If you want to play a mod pack you can press "Browse Modpacks" and then a it will load by most downloads if you want to look for a new one you can use the search bar to the left of Create Custom Profile. If you do pick to Browse Modpacks it will show them up down throught the whole app when selected you can sort by the option box or filter by modpacks which will show you what type of modpack your more looking forward for
